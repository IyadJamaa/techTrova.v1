What Is Web Development

in general terms, it's the tasks associated with creating, building, and maintaining websites and web
applications that run online on a browser, web development is mainly separated into frontend and
backend development, there are many other roles that we will discuss in other paths.


In order to develop a website you mainly need:
• A User Interface (UI) design of your website.
• A front-end developer to turn that design into working code.
• A back-end developer to create the server and the database for that website.
• Tester




1. What Is Frontend

A Front-end developer is responsible for the client side (the client-side is everything that is
displayed on the user's device. Everything from the images and text displayed to the UI of a web
browser make up client-side. This is often referred to as the front-end of web development).
in short a front end developer is responsible for turning the design of a website (UI) which is
given to him by a UI designer and turning it into an actual interactive website using code.

2. What Is Backend

A back-end developer is responsible for the server side ( server side refers to the processing
that executes whatever the user is trying to do and makes the exchange of information
possible. This is where the data and source code are stored, and so it is often referred to as the
back-end. The server-side processes user requests on a web page and sends information back
to the client. It is used for anything that requires dynamic data, such as login information,
running payments, pulling live data, sending emails, and other complex functions.)
in short a back-end developer is responsible for creating the server and the database for the
website and more.
